Atlas Terrain System

Collaboration diagram for Atlas Terrain System:

Detailed Description

Atlas is a flexible framework for paged terrain.


Modules

 Atlas Core
 Core Atlas classes and functionality.
 Atlas Editing
 Functionality related to editing and generating Atlas terrains.
 Atlas Resource System
 Although the core background loading system is implemented in AtlasCore, the specific data types that are loaded, the rules for regenerating them, and so forth all live in this group.
 Atlas Runtime
 Functionality that relates to rendering and interacting with an Atlas terrain in a running game instance.
