Detailed Description

Allow traversal over atlas geometry using UV coordinates.

Performs normal interpolation.

Perform a lookup in UV space.

Takes a UV parameter, and returns the index of the triangle it's on, the position in 3space at which that UV is located, and a smoothed normal for that location.
