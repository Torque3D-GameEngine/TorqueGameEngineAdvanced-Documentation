Revision History.

0.6.3.0 : Added PreferencesUpdated() and ShutdownRequested(). Added FullScreen, Widget, and Game-rendered Menu methods. All RPC-related methods changed to return GGCError instead of void. All methods changed from pure abstract to default implementation. 0.6.1.1 : Added constructor and virtual destructor. 0.6.1.0 : Added DynamicOwnershipFailed(...) : Added Fullscreen*() Methods. 0.6.0.0 : Added virtual GetVersion() (default returns NULL) : Added DefaultCommandHandler(). : Added OwnershipRefreshed(). 0.5.0.12 : Updated GameInit to take 'dedicated' argument. Updated HostStart() and SpStart() to return a GGCError. 0.5.0.7 : Added BlurSession, FocusSession, and SetFocus methods. 0.5.0.1 : Created.
